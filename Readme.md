计算机程序的构造和解释习题
======================
希望使用SICP这本书作为函数式语言学习的一个开端。

MIT/GNU Scheme
--------------
使用版本9.2，按官方文档说明安装

### 基本使用

####退出
```
(exit)
```
退出scheme会出现一句提示语: Moriturus te saluto. 是一个拉丁短语，意为：Those who are about to die salute you。
http://en.wikipedia.org/wiki/Ave_Imperator,_morituri_te_salutant

### 加载文件
(load "path/to/.scm")
(pwd) 当前目录
(cd "path") 切换目录
